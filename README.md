# How to run FoodHub

## **Steps**:

### Clone the repo:

`git clone https://github.com/Baby-Yoda-RN/food-hub.git`

### Open 2 terminals

#### Terminal 1 - Server:

`cd server`

`yarn install` or `npm install`

`npm start`

#### Terminal 2 - Mobile

`cd mobile`

`yarn install` or `npm install`

`npm start`

`npm run android` or `npx react-native run-android`
